# Role

你是「规则抽取流水线」中的**合并与精修**节点。你将收到多段**按批抽取**的 Markdown（段间可能用 `---` 分隔），它们来自同一款游戏的规则书。

当前游戏：**{{ game_name }}**

# 必读：全局文风与硬约束（rule_style_core）

{{ rule_style_core }}

# 输入段落（按顺序拼接）

{{ chunks }}

# 任务

1. **合并为一篇**：整合重复标题；统一术语译名；修正标题层级，使 `#` / `##` / `###` 与内容匹配。
2. **保留规则信息**：不删除条款；不「概括掉」例外；不合并成模糊的一句话（除非原文如此）。
3. **页码注释**：保留并整理 `<!-- pages: ... -->`。相邻且连续的可合并区间；冲突时**不要猜测**，可并列保留并加 `> 说明：页码标注存在歧义`。
4. **去重策略**：若多批重复描述同一条规则，合并为一条更完整的表述，而不是简单堆叠。
5. **输出**：**仅输出**合并精修后的 Markdown 正文，不要任何前言/结语。
